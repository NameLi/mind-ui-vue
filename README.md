# Mind UI Vue

> Mobile UI for **Vue 2.6**

## 警告
本组件库仅作为个人学习与项目使用，未经过严格测试，请勿在生产环境环境中使用。

## 演示
[演示地址](https://mind-ui-vue.ixook.com)


## 文档
[使用文档](https://mind-ui.ixook.com/docs)

## 安装
```shell
npm i mind-ui-vue -S
```

## 使用
全量引入

```javascript
import Vue from 'vue';
import MindUI from 'mind-ui-vue'
Vue.use(MindUI)
```

## 开发环境

```shell
npm run serve
```
